# Spotify Top Hits — Exploratory Data Analysis

## Overview
EDA on Spotify Top Hits dataset (1975-2021) using Python.

## Questions Answered
- Which genres dominate the top hits?
- Which artists appear most?
- What makes a song popular?
- Is there a relationship between energy and danceability?
- How has popularity trended over time?

## Tools Used
- Python, Pandas, Seaborn, Matplotlib
